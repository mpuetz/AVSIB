HINWEIS
Sollten Sie einen Fehler auf Grund einer fehlenden SQL-Server Installation erfahren können Sie unter 
dem Installationspfad eine Datei mit Namen SQLServerLocalDB.msi finden, 
welche die benötigten Funktionen auf ihrem System installiert.

CREDITS:
Das Drucken der ListBox basiert auf einer Klassen von Matthias Zürn:
